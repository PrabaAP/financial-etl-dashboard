## What this project does

Takes messy financial exports (simulating a real QuickBooks / Sage 50 client) and automatically turns them into a live, interactive executive dashboard — in 5 Python scripts.

**Target use case:** Mid-market accounting firms that need to bridge the gap between legacy systems (Sage, QuickBooks, Excel) and modern automated analytics.

---

## Pipeline overview

```
Raw messy CSVs  →  Python cleaning (pandas)  →  SQLite database
     →  SQL analytics queries  →  Streamlit executive dashboard
```

| Script | What it does |
|--------|-------------|
| `01_generate_data.py` | Creates realistic messy General Ledger + Accounts Receivable CSVs |
| `02_clean_data.py` | Cleans dates, amounts, account codes, duplicates, blanks |
| `03_load_database.py` | Loads clean data into a local SQLite database |
| `04_analytics.py` | Runs SQL queries: MoM variance, 3-month rolling avg, AR aging |
| `05_dashboard.py` | Streamlit dashboard with KPI cards and Plotly charts |

---

## Dashboard features

- **KPI cards** — Total Revenue, Total Expenses, Net Cash Flow, Outstanding AR
- **Monthly Revenue** — bar chart by month
- **MoM Revenue Variance** — colour-coded (green = growth, red = decline)
- **Net Cash Flow + 3-Month Rolling Average** — line chart with trend overlay
- **AR Aging Buckets** — 0-30 / 31-60 / 61-90 / 90+ days outstanding
- **Expense Breakdown** — stacked bar by category per month
- **Top Customers** — ranked by outstanding AR balance

## Setup & run

### Prerequisites
- Python 3.9 or higher
- Mac, Windows, or Linux

### Install dependencies
```bash
pip install -r requirements.txt
```

### Run the full pipeline (in order)
```bash
python scripts/01_generate_data.py
python scripts/02_clean_data.py
python scripts/03_load_database.py
python scripts/04_analytics.py
streamlit run scripts/05_dashboard.py
```

The dashboard opens automatically in your browser at `http://localhost:8501`.

## Deploy to the web (free)

1. Push this folder to a public GitHub repository
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub account
4. Select this repo → `scripts/05_dashboard.py` as the main file
5. Click **Deploy** — you get a shareable public URL
